# Adaptive Hospital Readmission Decision Support System

[![DOI](https://zenodo.org/badge/1302103533.svg)](https://doi.org/10.5281/zenodo.21385508)

## Overview
The Adaptive Hospital Readmission Decision Support System Framework uses machine learning to estimate the probability of readmission. 
The framework also provides clinical decision support through providing the influential features at the population level, providing 
care recommendations and allowing users to assess the impact of various interventions on the readmission risk through a simulator.
